Défi : Fonctionnalités HTML-CSS-JS sans librairie JS ou CSS

Création de fonctionnalités d'éléments HTML-CSS-JS de page web sans faire usage de librairie JS ou CSS.


Ce projet consiste de trois Étapes :


Étape 1 :


Dans un premier temps, une liste suivante affichée dans une liste à puce HTML:


• Fruits rouges

• Raisins blanc

• Pomme verte

• Haricots verts

• Haricots rouges

• Rouges gorges

• Fruits exotiques rouges et verts


Étape 2 :


Dans un second temps, faire une page HTML sur laquelle il y aura 5 boutons. Chaque bouton réalisera une action précise.


1. Bouton 1: Lorsqu’on clique dessus, il changera de manière aléatoire la couleur du fond de la page


2. Bouton 2: Lorsqu’il est survolé, tous les autres boutons disparaissent et il y a une animation fluide du bouton qui se projette en avant (comme si on zoomait dessus et qu’il devenait plus gros)


3. Bouton 3: Lorsqu’on clique dessus, la chose suivante ce produit: a. Le libellé du bouton est remplacé par “Début dans…” b. Puis un décompte démarre de 10 à 0 seconde c. Å 0, le libellé du bouton est remplacé par les coordonnée X et Y du pointeur de la souris sur l’écran au moment du clic.


4. Bouton 4: affichera que les occurrences de la liste affiché sur la page dont le nom fini par “rouges”


5. Bouton 5: reinitialise le filtre du 5ème bouton si il a été utilisé et tri la liste dans l’ordre alphabétique (La liste étant de base dans un ordre aléatoire)


Étape 3 :


Enfin, un champ sera présent sur la page et permettra à l’utilisateur d’entrer son prénom. Le prénom s’affichera dans une box en bas à droite de l’écran (quelque que soit la taille de l’écran) dans la phrase suivante: Bonjour [prénom] et bienvenue sur le site !


Aucune librairie JS ou CSS ne devra être utilisé.
